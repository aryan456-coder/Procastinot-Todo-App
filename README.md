Procastinot 🚀

Turn procrastination into productivity, one task at a time.

Procastinot is a modern and intuitive task management application designed to help users organize, prioritize, and track their daily tasks efficiently. With a clean interface and seamless user experience, staying productive has never been easier.

🌐 Live Demo

Try it here:
https://aryan456-coder.github.io/Procastinot-Todo-App/

✨ Features
➕ Add new tasks instantly
✅ Mark tasks as completed
🗑️ Delete tasks when finished
✏️ Edit existing tasks
📂 Organize and manage your workflow
💾 Persistent task storage
📱 Responsive design for desktop and mobile
🎨 Clean and modern UI
🛠️ Built With
HTML5
CSS3
JavaScript (ES6)
🚀 Getting Started
Clone the repository:
git clone https://github.com/aryan456-coder/Procastinot-Todo-App.git
Navigate to the project folder:
cd Procastinot-Todo-App
Open index.html in your browser.

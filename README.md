# 🚀 Procastinot

### Turn Procrastination into Productivity, One Task at a Time.

A modern and feature-rich task management application that helps users organize, prioritize, and track their daily tasks with an intuitive and visually appealing interface.

🌐 **Live Demo:**  
https://aryan456-coder.github.io/Procastinot-Todo-App/

</div>

---

## ✨ Features

✅ Create, edit, and delete tasks

🎯 Set task priorities (High, Medium, Low)

📅 Add due dates and manage deadlines

📂 Organize tasks with categories

📊 Real-time analytics and productivity insights

🌙 Dark & Light mode support

🔍 Search and filter tasks instantly

📈 Progress tracking dashboard

💾 Local storage persistence

📱 Fully responsive design

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/aryan456-coder/Procastinot-Todo-App.git
```

### Navigate to Project Folder

```bash
cd Procastinot-Todo-App
```

### Run the Application

Simply open:

```bash
index.html
```

in your browser.

## 🎯 Why Procastinot?

Most to-do applications focus only on task creation.

**Procastinot** goes beyond that by helping users:

- Stay organized
- Prioritize effectively
- Monitor progress visually
- Build productive habits
- Reduce procrastination
